# C33 · Crunch SQL — Brand Guide

- **Accent:** Code Crunch blue `#2463EB` (with deep `#1d4ed8`).
- **Voice:** rigorous and demystifying. Always show the query, then the plan, then *why* the database chose it. Nothing is magic — the planner is a cost model you can read.
- **Icon motif:** a table + an index B-tree.
- **One-liner:** "From your first SELECT to reading query plans like a senior."
